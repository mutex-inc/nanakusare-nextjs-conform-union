## 準備

### node.js のバージョン管理ツール

node.js のバージョン管理ツールとして `Volta` を利用します。以下より、インストールしてください。

- [`Volta`](https://volta.sh/)

### 依存関係のインストール

```bash
yarn
```

## 開発

### local 環境での起動

```bash
yarn dev
```
